## О репозитории

В этом репозитории находятся описание и реализации лабораторных работ в рамках
курса по Software Design в магистратуре "Программное обеспечение высоконагруженных систем"
Университета ИТМО.

## Структура репозитория
Лабораторные работы расположены в поддиректориях, названия которых соответствуют
названиям задачи. В каждой из таких директорий в отдельном `README.md` описана суть и условия.

Также в директории `practice` находятся задачи с практических занятий на построение диаграмм.

## Список лабораторных работ
- `shell` — дизайн архитектуры и реализация оболочки командной строки
    - Подзадача: реализация `grep` в виде встроенной команды
- `chat` — реализация простого чата с клиент-серверной архитектурой
